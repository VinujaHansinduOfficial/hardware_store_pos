# Hardware Store POS - PHP + MySQL

## Setup
1. Create a folder in your XAMPP/htdocs named `hardware-store-pos`
2. Copy all project files into that folder
3. Import `database.sql` into phpMyAdmin
4. Update `config/database.php` with your MySQL username/password
5. Open: `http://localhost/hardware-store-pos/`

## Notes
- Delete is handled as `is_active = 0`
- This is a starter structure based on your requested folder tree
- You can extend each module with validations, joins, and proper AJAX flows
